Cazó **6 bugs críticos** silenciosos en Planta · todos cerrados sin
regresión (117/117 GP verdes al final):

- **B1** `inventario.py:5811` planta_stock_por_lote agrupaba por
  (lote, fv, estado) separando Entradas de Salidas FEFO · Bodega
  Materias Primas mostraba stock viejo (1000g - 300g consumidos
  seguía mostrando 1000g). Fix: GROUP BY solo por lote + MAX(CASE WHEN
  tipo='Entrada' THEN ...) para fv/estado.
- **B2** `inventario.py:3708` conteo_ajustar NO chequeaba
  `ajuste_aplicado` al entrar · doble-click del operario o reintento
  duplicaba INSERT. Fix: guard 409 + atomic claim
  UPDATE-WHERE-COALESCE(ajuste_aplicado,0)=0.
- **B3** `programacion.py:4866` _distribuir_fefo SIEMPRE marcaba
  sin_lote=True cuando remainder>0.01 · race condition entre
  producciones paralelas insertaba sin_lote y ocultaba stock fantasma
  negativo. Fix: validación previa de stock disponible antes de la
  marca sin_lote.
- **B4** `inventario.py:5012` mee_import_bulk pierde signo del ajuste.
  Antes insertaba `tipo='Ajuste'` con abs(delta) · stock_mee_calculated
  trataba Ajuste como +cantidad · drift permanente de 2×|delta|. Fix:
  usar Entrada/Salida según signo del delta.
- **B5** `inventario.py:2524` eliminar_lote hard DELETE de TODOS los
  movs incluido Salidas FEFO consumidas · rompía trazabilidad INVIMA
  Resolución 2214/2021 art. 10. Fix: contra-movimiento del saldo neto
  (Salida si saldo>0, Entrada si saldo<0) con
  `estado_lote='ELIMINADO'`. Historial preservado.
- **B6** `inventario.py:3470` threshold gerencia 5% se evadía cuando
  stock_sistema=0 (`if stock_sis>0 else 0` → pct_diff=0). Fix: si
  stock_sis=0 y diff≠0 → pct_diff=1.0 forzado → requiere_gerencia=1.

## Reglas nuevas decididas

1. **numero_op format congelado**: `OP-YYYY-NNNN` (4 dígitos
   zero-padded). Cambiar el format rompería import legado MyBatch y
   etiquetas físicas en planta. Versionar via mig si jamás se necesita
   cambiar.

2. **codigo_pt es opcional**: NULL aceptado para evitar bloquear flujo
   actual donde la mayoría de productos NO tienen mapeo MyBatch. UI
   eventualmente alertará al usuario que un producto sin codigo_pt no
   se puede importar al legado.

3. **zona='controlada'** es el default para áreas de manufactura HHA.
   Si se agrega manufactura aséptica futura, usar 'limpia' explícito.

4. **Operación delete-restore en eliminar_lote**: NO hacer hard DELETE
   de movimientos cuando hay Salidas FEFO consumidas · usar
   contra-movimiento con estado_lote='ELIMINADO'. INVIMA Res 2214/2021
   art. 10 exige preservar trazabilidad.

5. **Threshold gerencia 5% con stock_sistema=0**: forzar
   requiere_gerencia=1 cuando hay diferencia física pero no había
   stock registrado. Evita bypass del control BDG-PRO-002 num 8.
