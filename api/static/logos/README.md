# Logos HHA Group

Coloca aquí los logos PNG (con fondo transparente) para que la app los embeba en
PDFs, headers y emails:

| Archivo | Uso | Recomendado |
|---|---|---|
| `hha_group.png` | Logo paraguas grupo empresarial | 600x600 px |
| `animus_lab.png` | Logo ÁNIMUS Lab | 600x600 px |
| `espagiria.png`  | Logo Espagiria Laboratorio | 600x600 px |

Si un archivo no existe, la app usa header de texto (no falla).

## Paleta oficial ÁNIMUS Lab (extraída del manual de identidad v1)

- **Negro principal**: `#000000`
- **Blanco**: `#FFFFFF`
- **Neutrales** (cremas/beige): `#F7F4EF` · `#EDEAE3` · `#DFD7CD`
- **Acentos**: `#B4C1E2` (azul polvo) · `#E0BAA8` (terracota) · `#D1B9B4` (rosa) · `#9B997B` (oliva) · `#444444` (gris oscuro)
- **Acabados**: Pantone Gold 871 C, Pantone Silver 877 C

## Tipografías oficiales

- **Títulos**: Banana Grotesk
- **Texto**: League Spartan (Google Fonts)
- **Acentos**: Instrument Serif (Google Fonts)

Para PDFs usamos Helvetica como fallback (PDF estándar). Para web podemos
cargar League Spartan + Instrument Serif desde Google Fonts.
